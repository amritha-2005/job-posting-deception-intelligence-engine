05 — Methodology
How It Works
01
Data & Features
18,238 job postings from the Kaggle Real/Fake Job Posting dataset. 13 engineered features: keyword signals, caps ratio, salary width, logo presence, exclamation count, unique word ratio.
02
Deception Taxonomy
Pioneer layer — each posting is labeled across 5 deception archetypes simultaneously: scam, ghost job, bait & switch, discriminatory, and vague trap. First multi-label deception classifier of its kind.
03
Ensemble Classifier
Soft-voting ensemble: Random Forest (200 trees, depth 15) + Gradient Boosting (150 estimators, lr 0.08) + Logistic Regression. TF-IDF on 5,000 bigrams combined with numeric features.
04
Cluster Universe
All 866 fake postings reduced to 50 dimensions via TruncatedSVD, then projected to 2D using t-SNE (perplexity=30). K-Means (k=5) identifies deception clusters mapped to the taxonomy.
05
Company Fingerprinting
Each company receives a deception score across all 5 types aggregated from its posting history. Companies ranked by fraud rate — exposing systematic patterns in employer deception.
06
NLP Explainability
Logistic Regression coefficients mapped to the TF-IDF vocabulary create an interpretable word-level X-ray of manipulation language — showing exactly which words drive each prediction.
